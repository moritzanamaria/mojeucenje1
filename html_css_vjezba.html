HTML Elementi

Osnovna struktura dokumenta

<html>
Korijen cijelog HTML dokumenta. Sve ide unutar.
<html lang="hr"> ... </html>
<head>
Meta-informacije, title, linkovi na CSS, skripte. Nije vidljivo u browseru.
<head><title>Moja stranica</title></head>
<body>
Sav vidljivi sadržaj stranice.
<body> ... sav sadržaj ... </body>
<title>
Naslov u browser tabu i bookmarkovima.
<title>Moja web stranica</title>
<meta>
Metadata: charset, viewport, opis za SEO. Samozatvarajući tag.
<meta charset="UTF-8">
<meta name="description" content="...">
<link>
Povezuje vanjski CSS ili fontove.
<link rel="stylesheet" href="style.css">
<style>
CSS direktno unutar HTML dokumenta (u <head>).
<style> p { color: red; } </style>
<script>
JavaScript kod ili link na JS datoteku.
<script src="app.js"></script>
