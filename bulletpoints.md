# 📘 Little guide

Popis pojmova koje sam naučila na predavanjima.

### 🕒 Što je verzioniranje koda (Version Control)?
* **Verzioniranje koda** - Sustav koji prati povijest promjena u datotekama. Radi kao "Time Machine" za programere.
* **Zašto ga koristimo:**
  - **Nema straha od grešaka:** Ako napišem kod koji sve pokvari, mogu vratiti stariju verziju koja je radila.
  - **Nema duplanja mapa:** Imamo samo jednu mapu, a Git pamti sve njezine prošle verzije.
  - **Praćenje povijesti:** Točno se vidi tko je, kada i što promijenio.

### 💻 Pokretanje File Explorera i Command Prompt-a (Windows)

#### 📂 File Explorer (Žuta mapa)
* **Prečac:** Pritisni **Windows tipka + E** na tipkovnici za trenutno otvaranje.

#### 🖲️ Command Prompt (Naredbeni redak / Command centar)
* **Prečac:** Pritisni **Windows tipka + R**, upiši **`cmd`** i pritisni Enter.
* **Start meni:** Pritisni Windows tipku, upiši **"cmd"** (ili "Command Prompt") i pritisni Enter.

  
### 🐙 Git & GitHub Pojmovi
* **Git** - Alat na računalu koji radi kao vremenska kapsula i pamti svaku izmjenu datoteka.
* **GitHub** - Web stranica na internetu gdje šaljemo i čuvamo svoje projekte.
* **GitHub Desktop** - Program s grafičkim sučeljem (gumbima) na računalu koji ti olakšava korištenje Git-a bez tipkanja naredbi u terminal.
* **Repository (Repozitorij)** - Mapa na GitHubu ili računalu u kojoj se nalaze dokumenti projekta.

### 🛠️ Git Naredbe
* **`git clone`** - Kopira projekt s interneta na računalo.
* **`git commit`** - Radi "snimku stanja" i sprema promjene na računalu.
* **`git push`** - Šalje sve spremljene promjene s računala gore na GitHub.
